Embedding Live Charts

For example, the leaflet (Cheng, Karambelkar, and Xie 2017) package is great for producing interactive maps. Using the following example taken from here, we can create maps in only a few lines of code:

library(leaflet)

m <- leaflet() %>%
  addTiles() %>%  # Add default OpenStreetMap map tiles
  addMarkers(lng=-1.396, lat=50.93, popup="The birthplace of R")
m

Cheng, Joe, Bhaskar Karambelkar, and Yihui Xie. 2017. Leaflet: Create Interactive Web Maps with the Javascript ’Leaflet’ Library. https://CRAN.R-project.org/package=leaflet.
